# 📝 Sign Up Form

Um projeto de **formulário de cadastro responsivo** desenvolvido utilizando **HTML5 e CSS3**.
O objetivo do projeto é praticar a criação de formulários, organização semântica do HTML e estilização moderna com CSS.

---

## 🚀 Funcionalidades

* Campo para **nome**
* Campo para **email**
* Campo para **senha**
* Seleção de **faixa etária**
* Campo de **biografia**
* Seleção de **cargo profissional**
* Seleção de **interesses**
* Botão para envio do formulário

---

## 🛠️ Tecnologias Utilizadas

* **HTML5**
* **CSS3**
* **Google Fonts**

Fontes utilizadas:

* Nunito
* Open Sans
* Raleway
* Roboto

## 🎨 Layout

O formulário possui:

* Layout centralizado na tela
* Fundo suave para melhor leitura
* Campos de entrada estilizados
* Botão com efeito **hover**
* Separação de informações usando **fieldset** e **legend**
